Access CVL@UWA

The cluster is accessible via https://desktop.cvl.org.au/strudel-web/#/system-selector (CVL@UWA dropdown)

Join an existing project

  1. Login to https://hpc.erc.monash.edu.au/aafbootstrap with AAF credentials.
  2. Click on “Join existing project” and enter project ID.

Create a new project

  1. Login to https://hpc.erc.monash.edu.au/aafbootstrap with AAF credentials.
  2. Fill in the New Project Form.

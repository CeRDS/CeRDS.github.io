CVL@UWA

Overview

CVL@UWA is a part of the Characterisation Virtual Laboratory National project. It is hosted at the Pawsey Supercomputing Centre primarily within the Nimbus Cloud Service.

This page outlines the UWA implementation of CVL housed at Pawsey. It contains details about design decisions, workflows, tools and other miscellaneous material.
